2. Основные тэги в документе

1. Добавьте в документ несколько анкоров (внутренний метки).

2. Добавьте в документ несколько ссылок.

3. Добавьте в документ несколько картинок, каждая из которых с уникальным идентификатором, задайте ширину и высоту.

4. Выведите с помощью JS кол-во анкоров

5. Выведите с помощью JS кол-во ссылок

6. Покажите содержимое (innerHTML) первого анкора на странице

7. Выведите количество картинок

8. Выведите ширину первой картинки

9. Выведите ширину самой широкой картинки

10. Выведите сумму всех высот картинок
